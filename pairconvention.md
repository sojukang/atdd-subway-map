# 라라&소주캉 페어 컨벤션
## 페어 프로그래밍
- 10분 마다 드라이버 - 네비게이터 역할 변경
- 40분 개발 - 20분 휴식
- 테스트 코드 없이 프로덕션 코드 작성하지 않기

## 소셜  
- 비난하지 않기
- 박수 많이 쳐주기(소주캉)
- 다른 페어의 외란에 흔들리지 않기 
- 우리 코드가 최고다!!

## 코드 컨벤션
- getter 형 메서드는 get을 붙이기
- static factory method는 필요할 때 사용하기 
- method 순서 
  - 쓰이는 메소드를 순차적으로 읽을 수 있게 응집시키기
- 최대한 접근 제어자를 좁게 유지하여 객체를 은닉하기
